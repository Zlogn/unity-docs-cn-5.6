Windows Store Apps: Association launching

通过 API 可针对文件类型或协议启动用户的默认应用程序。请参阅 UnityEngine.WSA.Launcher 以了解更多信息。

还可将应用程序设置为某个文件类型或协议的默认应用程序。在 Player Settings 中即可设置此信息（更多设置位于 Package.appxmanifest > Declarations 中）
