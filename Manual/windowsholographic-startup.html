运行您的第一个应用程序

在 Unity 中构建项目

1.配置内容。像往常一样创建 Unity 项目。 2.为 HoloLens 配置 File > Build Settings。请参阅设置以了解这些设置的详细信息。 3.在 Unity 中构建项目。成功构建会生成一个 Visual Studio 解决方案，其中包含您的项目以及在设备上构建和运行所需的所有内容。此过程应该会自动打开一个包含解决方案路径的操作系统窗口。

从 Visual Studio 中运行

1.打开上面步骤 3 中生成的解决方案。 2.指定设备详细信息。选择 Release、x86 和 Remote Machine。输入要运行的基础设备的 IP，并在 Authentication Mode 中选择 Universal (Unencrypted Protocol)。 3.在 Visual Studio 中构建解决方案。如果没有错误且目标设备未配对，则构建将继续，直到显示对话框窗口。对话框窗口要求提供配对号码。 4.与 HoloLens 设备配对： * 在设备上，选择 Settings > Update (Device update, reset, developer) > For Developers > Pair__，随后将打开一个包含号码的对话框窗口。 * 在 Visual Studio 对话框窗口中输入此号码。 * 当设备完成配对时，设备上包含配对编号的窗口应显示 Done__ 按钮。在此按钮上使用缩放点击手势继续：用一只闭合的手将拇指和食指按在一起，然后伸出拇指和食指。 5.部署将继续。应用程序现在应该会运行。

注意：应该只需要为每台机器、每台设备配对一次。
