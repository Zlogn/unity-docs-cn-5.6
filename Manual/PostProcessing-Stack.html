Post-processing stack（后期处理栈）

post-processing stack（后期处理栈）是一种超级 (über) 效果，将一整套效果组合到一个后处理管线中。这样做有一些优点：

  • 始终以正确的顺序配置效果

  • 允许将多种效果组合成一个 pass

  • 所有效果都在 UI 中分组在一起，因此提供更好的用户体验

Post-processing stack(后期处理栈)还包括一组监视器和调试视图，旨在帮助您正确设置效果并调试输出中的问题。

You can download the post-processing stack from the Asset Store.

For help on how to get started with the post-processing stack see Setting up the post-processing stack.

效果

  • 抗锯齿（FXAA 和 TAA）
  • 环境光遮挡
  • 屏幕空间反射
  • 雾效
  • Depth of Field
  • 运动模糊
  • 人眼适应
  • 泛光
  • 颜色分级
  • 用户 LUT
  • 色差
  • 颗粒
  • 渐晕
  • 抖动

有关Post-processing stack（后期处理栈）中包含的每个单独效果的详细信息，请参阅该效果的页面。
