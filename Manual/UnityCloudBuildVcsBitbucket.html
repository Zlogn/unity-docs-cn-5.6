将 Bitbucket 用于 Unity Cloud Build

Unity Cloud Build 可以连接到 Bitbucket (bitbucket.org) 上托管的 Git 和 Mercurial 代码仓库。

使用 Bitbucket 上的公开 Git 代码仓库

Unity Cloud Build 可以自动连接到 Bitbucket 上的公开代码仓库。在 Cloud Build 中设置项目时，只需复制并粘贴代码仓库的 URL。

使用 Bitbucket 上的私有 Git 代码仓库

如果您的 Bitbucket 代码仓库为私有仓库，请查找在项目的 Cloud Build 设置期间由 Unity 提供的 SSH 密钥，然后按照以下说明将其添加到您的 Bitbucket 帐户。

1.在 bitbucket.org 上登录您的 Bitbucket 帐户

2.打开用户菜单，然后单击 Manage Account

3.从左列菜单中选择 SSH Keys

4.单击 __Add Key__，然后为其添加标签（例如 “Unity Cloud Build”）

5.将 Unity Cloud Build SSH 密钥粘贴到 Bitbucket Key 字段中

6.单击 Add Key

使用 Bitbucket 上的 Mercurial 代码仓库

请参阅有关将 Mercurial 用于 Unity Cloud Build 的文档以了解详细信息。
